ADD DESCRIPTION

Quick overview

Spring configuration file spring-configurations/freme-ner.xml
Source code location freme ner source code
Packages that contain freme ner freme-ner-dev

Configuration options

Freme ner is configured using application properties, e.g. in the configuration file application.properties. The configuration has this form:

freme.ner.sparqlEndpoint=
freme.ner.solrURI=
freme.ner.languages=en,de,nl
freme.ner.modelsLocation=C:/path/to/the/model/files/folder
freme.ner.domainsFile=src/main/resources/domains.csv

## database settings for dataset metadata handling: move to freme-ner-dev? (like broker-dev)
#spring.datasource.url=jdbc:mysql://localhost:3306/freme
#spring.datasource.username=
#spring.datasource.password=
#spring.datasource.driver-class-name=com.mysql.jdbc.Driver
